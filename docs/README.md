## Run locally
Open `index.html` in your browser (or use a simple static server).

Example (Python):
```bash
python3 -m http.server 5173
```
Then visit `http://localhost:5173`.

## Customization (quick)
- Business name + contact info (site-wide): edit `js/main.js` (site config).
- Colors/typography: `css/styles.css` (`:root` variables).
- Pricing tiers + deliverables: `pricing.html`.
- Logo: navbar uses `assets/velora-logo.png` (place your logo image at this exact path).
